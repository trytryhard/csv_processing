# Пример использования скрипта
```
python .\src\main.py --files .\test\test_data\corrupted_col.csv  .\test\test_data\corrupted_sep.csv .\test\test_data\fine_file.csv .\test\test_data\fine_file2.csv --report sum-population --groupby year continent --v 1 --encoding utf8 --d 1
```

# Разбор флагов 
## `--files`\\`-f`
Принимает путь(-и) до csv-файл(-ов)    

## `--report`\\`-r` 
Принимает аргумент с названием вида расчета, отделенный одним дефисом от названия колонки, по которой необходимо выполнить расчет.  
Пример `--report sum-king-col`, здесь будет расчет суммы по колонке `king-col`.  Расчет выполняется, учитывая столбцы из `--groupby`.  
Возможные виды расчета:
- sum - рассчитывает сумму по переданному столбцу в аргументе
- avg - рассчитывает среднее значение по переданному столбцу в аргументе
- count - рассчитывает количество повторений элемента по переданному столбцу в аргументе
- mode - рассчитывает моду элемента по переданному столбцу в аргументе
  

## `--groupby`\\`-gb` 
Группировка CSV-файлов по дополнительным колонкам. **Значение по умолчанию**, это колонка `country`.

## `--validation`\\`-v` 
Предварительная валидация csv-файлов на факт наличия колонок, фигурирующих в `--report` и `--groupby`.  
В ходе проверки исключает неподходящие файлы из дальнейшего расчета.   
Если передать значение "False" или "0", тогда предварительная валидация не будет применена.

## `--separator`\\`-s` 
Позволяет пользователю определить разделитель, который будет соответствовать анализируемым CSV-файлам. Если передан разделитель длиной больше 1 символа, то будет выбран первый символ строки.
Неверно указанный разделитель приведет к некорректным результатам работы программы.   
**Значение по умолчанию** - запятая.   

## `--encoding`\\`-e` 
Позволяет пользователю определить кодировку передаваемых CSV-файлов. Если будет передана несуществующая кодировка - программа уведомит ошибочный флаг.   
Неверно указанная кодировка приведет к некорректным результатам работы программы.  
**Значение по умолчанию**: UTF8  

## `--descending`\\`-desc` 
Позволяет выставить сортировку по рассчитываемому полю. по умолчанию сортировка выставлена по убыванию.
Если передать значение "False" или "0", тогда сортировка будет по возрастанию.

## `--help` 
Вызывает справку. Пример использования: `python main.py --help`
